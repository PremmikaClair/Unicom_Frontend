# Unicom 

Course project for Software Engineering, Kasetsart University  
Team project (12 members)

## About this project

Unicom is a university community application that helps students stay connected through
announcements, events, clubs, and other campus information.

This repository contains the **mobile application (Flutter)** part of Unicom.

- Mobile app (Flutter): this repository  
- Web application: developed by teammates   
- Backend API: developed by teammates 

## Tech Stack (Mobile)

- Flutter  
- Dart
  
## Key Features (Mobile)

- Post feed for browsing university-related posts, with basic filtering to find relevant content.  
- Events section for club announcements and university activities, where users can view details and register directly in the app.  
- Trending view that highlights popular hashtags and topics on campus.  
- Create-post feature that allows users to publish their own posts to the community.
- Profile page showing a userâ€™s own posts and personal profile details.
